## 12. Stripe Entegrasyonu (Yıl 2+)

### Neden Stripe?

- TR'de doğrudan henüz desteklenmiyor (2026 itibarıyla limited)
- Ama: Stripe Atlas ile US Inc. açılırsa global ödeme alımı kolay
- Şimdilik: **İyzico** (TR) + **Stripe** (AB/global, faz 3)

### TR İçin İyzico

```typescript
// src/services/iyzico.ts
import Iyzipay from 'iyzipay';

const iyzipay = new Iyzipay({
  apiKey:    process.env.IYZICO_API_KEY!,
  secretKey: process.env.IYZICO_SECRET_KEY!,
  uri:       'https://api.iyzipay.com',
});

export async function createSubscription(params: {
  customerId:  string;
  planCode:    string;
  cardToken:   string;
}) {
  return iyzipay.subscriptionInitialize.create({
    locale:           'tr',
    customerReferenceCode: params.customerId,
    pricingPlanReferenceCode: params.planCode,
    paymentCard:      { cardUserKey: params.cardToken },
  });
}
```

### Webhook Yönetimi

```typescript
// İyzico subscription event webhook
router.post('/webhook/iyzico', async (c) => {
  const signature = c.req.header('X-Iyzipay-Signature') ?? '';
  const body = await c.req.text();

  if (!verifyIyzicoSignature(body, signature)) return c.json({ error: 'Invalid' }, 401);

  const event = JSON.parse(body);
  switch (event.iyziEventType) {
    case 'SUBSCRIPTION_PAYMENT_SUCCESS':
      await activateSubscription(event.customerReferenceCode);
      break;
    case 'SUBSCRIPTION_PAYMENT_FAILURE':
      await markPaymentFailed(event.customerReferenceCode);
      // 3 başarısız → askıya al
      break;
    case 'SUBSCRIPTION_CANCELLED':
      await cancelSubscription(event.customerReferenceCode);
      break;
  }
  return c.json({ ok: true });
});
```

## 13. Davranışsal Anti-Pattern'ler (Yapılmayacaklar)

| Yapılmayacak | Neden |
|---|---|
| Banner reklam ekleme | Marka değerini yok eder |
| Kullanıcı verisi satma | Yasal + etik dışı |
| Aldatıcı dark pattern | Topluluk güvenini yok eder |
| Sponsorlu yorum/post sahnesi | Şeffaflık ihlali |
| Açıklamadan plan zammı | Kullanıcı kaybı |
| Free → Premium feature taşıma | "Bait and switch" |
| Lokasyona dayalı gizli fiyatlandırma | Etik dışı |
